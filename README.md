## ▪️ 어려웠던 점 & 아쉬웠던 점
**invert 기능** <br />
invert 버튼을 누르면 input 박스가 하나씩 비활성화 되는데 부정연산자를 사용해서 하는 과정이 아직은 헷갈리는 것 같았습니다.
<br /><br />
**박스 사이즈 트렌지션** <br />
박스의 height 값은 auto이기 때문에 단위를 선택하면 content의 값만큼 세로가 늘어납니다. 이 부분을 부드럽게 변하도록 하고싶었는데 아쉽게도 구현하지 못했습니다. CSS 실력을 좀 더 늘린 후 수정하도록 하겠습니다.
